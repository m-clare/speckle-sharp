## Developing & Debugging

### Requirements

- Visual Studio 2019 or above (or similar IDE)
- .NET Framework 4.8
- A Speckle account (you can make one at https://speckle.xyz/)

### Supported Versions

- ETABS v18, ETABS v19
- SAP2000 v21*
- CSiBridge v21*
- SAFE v20*

*Currently supported by the CSI API, but may not be fully implemented in Speckle yet!
